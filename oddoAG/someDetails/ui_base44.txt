Step dot (active):      w-8 h-8 rounded-full bg-primary text-primary-foreground ...
Step dot (pending):     w-8 h-8 rounded-full bg-muted text-muted-foreground ...
Connector line:         flex-1 h-0.5 bg-border mx-2
Animation
One custom keyframe powers all message entrances:
@keyframes slide-in {
  from { opacity: 0; transform: translateY(10px); }
  to   { opacity: 1; transform: translateY(0); }
}
/* Usage: */
.animate-slide-in { animation: slide-in 0.3s ease-out; }
